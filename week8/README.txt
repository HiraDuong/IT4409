SOURCE CODE ĐƯỢC LƯU TRONG FOLDER chat nên chỉ cần vào đó để chạy thôi
B1: Dùng VSCode trỏ vô folder chat (lưu ý phải vào tận folder chat)
B2: New Terminal chạy "nodemon server.js" hoặc "node server.js" đều được
B3: Truy cập localhost:3000 và chill

English:

Step 1: Open VSCode and navigate to the "chat" folder (make sure to go inside the "chat" folder).

Step 2: Open a new terminal and run either "nodemon server.js" or "node server.js."

Step 3: Access localhost:3000 in your browser and chill.
